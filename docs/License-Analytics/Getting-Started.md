# Getting Started

If you have not already installed the MLA app, please follow the [installation guide](Deployment/Standard-Install.md).

## Default Mode

1. Double click the desktop icon

2. Authenticate

## CLI Mode

1. Open PowerShell 7

2. Run `Get-LicenseCompliance`

3. Authenticate
